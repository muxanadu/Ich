ich. einrichten

Folge diesen Schritten einmalig, um die App auf allen deinen Geräten zu nutzen.

1Supabase-Konto erstellen
Gehe zu supabase.com → klicke Start your project → kostenloses Konto anlegen.

Erstelle ein neues Projekt (Name z.B. ich-dashboard, Region: Europe West). Warte bis das Projekt bereit ist (~1 Minute).
2Datenbank einrichten (SQL ausführen)
Gehe im Supabase-Dashboard zu SQL Editor → New Query → kopiere diesen kompletten Code und klicke Run:

→ Den SQL-Code findest du in der Datei supabase_setup.sql die zusammen mit dieser HTML-Datei geliefert wurde.
3API-Schlüssel eintragen
Gehe zu Project Settings → API.
Kopiere Project URL und anon public Key.

Öffne diese HTML-Datei in einem Texteditor und ersetze ganz oben im Script:
const SUPABASE_URL = 'https://DEINE-ID.supabase.co';
const SUPABASE_KEY = 'dein-anon-key';
const ALLOWED_EMAILS = ['deine@email.de', 'zweite@email.de'];
Speichere die Datei.
4Auf GitHub Pages veröffentlichen
1. Gehe zu github.com → neues Konto erstellen (kostenlos)
2. Klicke oben rechts + → New repository
3. Name: dashboard · Sichtbarkeit: Public · klicke Create repository
4. Klicke uploading an existing file → ziehe diese HTML-Datei hinein → Commit
5. Gehe zu Settings → Pages → Source: Deploy from branch → Branch: main → Ordner: /(root) → Save
6. Nach ~2 Minuten ist deine URL fertig: https://DEIN-NAME.github.io/dashboard/

Diese URL öffnest du auf iPhone, iPad, Mac, Windows — überall. Auf iPhone/iPad: Safari öffnen → URL eingeben → Teilen-Symbol → Zum Home-Bildschirm → wie eine App!
5Datei aktualisieren (bei Updates)
Wenn du eine neue Version der HTML-Datei bekommst:
Gehe zu deinem GitHub-Repository → klicke auf die alte Datei → Delete → lade die neue hoch. Fertig — URL bleibt gleich.
